##### **시스템 구축 전략**





###### **결측치 처리 / 표준정규화 / 주성분 분석 및 차원축소**



결측치 처리 시 수치형은 0으로 범주형/문자형은 '미상'으로 대체, 음수인 경우 해당 행 자체를 삭제

표준정규화를 통해 수치 상대화, 스케일 통일, 모델 효율성 향상, 이상치 영향 축소

1번 데이터의 인적정보 및 총 이용금액/이용건수 + 3번 데이터의 업종별 이용액 + 4번 데이터의 포인트/마일리지/할인액 + 할인율 위주로 컬럼 통합

이후 SVD -> 주성분 분석을 통해 차원축소 후 군집화에 변수로 활용

시계열 데이터 활용을 위해 증가율/감소율 변수 추가하거나 6개월치 전체 데이터 활용하되 최근 데이터에 더 큰 가중치 부여해야



###### **고객 군집화**



**Representative-based Voting 기법 기반**으로 군집화 실시
✔ 소수의 대표 고객군에 앙상블 클러스터링 적용

✔ 그 결과를 학습하여 전체 고객에게 빠르게 확장 적용



1. 고객 데이터 중 대표 샘플 층화추출
   -> 대표 샘플 수 : 2만명
2. 대표 샘플에 KMeans / MinMax KMeans / GMM / DBSCAN 등 사용하여 군집화

3\.  Agglomerative로 메타 군집화

    -> 계층적 관계·군집 내 서브그룹 존재 파악, 군집 간 유사성과 상하위 구조 해석에 효과적, 계산 비용 증가하나 대표 샘플링 통해 억제 가능

4\.  군집을 분류기에 학습시킨 후 분류로 군집 전체 데이터에 군집 할당

5\.  이후 일정 주기마다 재군집화하여 군집 데이터 최신화



**최적 군집 개수**의 결정도 가능

DBCAN 기법을 제외하고 군집화 시에 아래 지표 사용하여 군집을 평가할 수 있음

Gap Statistic: "군집 구조 자체가 유의미한가?"를 통계적으로 감별

Stability Criterion: "그 구조가 미래에도(데이터 변화에도) 일관적으로 재현되는가?"를 검**증**



1. 대표 샘플에 대해 K별 Gap, Stability 동시 계산

2\.  Gap 값이 최대이거나, 기준 오차 이내(오차 틀 내 큰 Gap).

3\.  해당 후보 K 안에서 Stability(ARI/NMI 등)가 가장 높은 K를 선택.



속도 및 비용의 효율성을 고려하였을때 위와 같은 방법으로 **카드 데이터의 군집화**도



###### **추천 모델**



**컨텐츠 기반 필터링 + 협업 필터링 기반 하이브리드 추천 시스템**



우선 블랙리스트 등재 고객의 경우 추천 대상에서 제외

**컨텐츠 기반 필터링**의 경우 고객 데이터와 카드 데이터의 코사인 유사도(두 벡터의 방향이 얼마나 일치하는 가?) 계산

카드 혜택 데이터 vs 고객 소비패턴 데이터(3번 데이터)

카드 포인트/혜택/마일리지 데이터 vs 고객 포인트/혜택/마일리지 데이터(4번 데이터)

카드 연회비 데이터 vs 고객 연회비 청구 데이터

정규화된 데이터의 유사도가 높을수록 코사인 유사도 증가

**정량적 평가지표**로 *기존 카드대비 혜택율 증가치 \* 코사인 유사도* 를 활용하여 고객의 혜택 만족도 평가, 가장 높은 카드 우선적으로 추천



**협업 필터링**의 경우 기존 고객 군집을 활용, 각 카드의 혜택 속성을 고객이 속한 군집의 대표 소비/혜택 패턴 벡터와 비교

이후 컨텐츠 기반 필터링 방법 사용하여 해당 군집의 대표 카드를 선정



신용+체크카드 월평균 사용액, 카테고리별 사용 비중, 청구혜택수, 혜택민감도 등 고객 데이터 신뢰도/충성도 지표가 높으면 높을 수록 **컨텐츠 기반 필터링 추천 카드 비중 ↑**

낮으면 낮을 수록 **협업 필터링 추천 카드 비중 ↑**

고객 데이터 신뢰도/충성도에 해당하는 컬럼 중 한 지표라도 -1.5 이하, 또는 총합 평균이 -1 이하이면 신규 고객으로 간주 **연령/성별/생애주기 고려 최소한의 대표카드만 추천**

**EX)**

신규 고객 유형	          추천 전략 (심리·사회학 기반)

20대 = 대학생/신입          무실적·무연회비, 온라인·간편결제 강점 카드

30대 = 맞벌이 신혼부부     교통·카페·쇼핑 자동적립, 실물+디지털 복합형

40대 이상 = 가구주	          마트/주유/보험·공과금 할인형, 가족카드 추가

‘기타’(정보 미흡)	          연회비 0원/저가, 실적조건 최소 대표카드



인지부하·선택과부하 방지 위해 **최종 추천 카드의 수는 3~5개 유지**, 기본적으로는 3개를 추천하되 [기준](https://www.perplexity.ai/search/nayi-mal-hoeweonjeongbo-gijunn-B12gByQqTCaWam5EL9ETEA#13)에 따라 최대 5개까지 확장
